《牧场物语：来吧风之集市》赚钱思路的分享

发布时间: 2025-10-05
最后更新: 2026-01-13
阅读时长: 22 分钟

牧场物语这个系列已经和我一样老了，虽然我玩的很早，从GBA时代就已经玩了牧场物语2和3，但是后续作品中也只体验了矿石镇。小时候没有渠道和资源去买nds和3ds，倒是很幸运的避开了Marvelous的很多摆烂作品。相比于M社的风评，风之集市这座的质量确实还能称得上优良，算是一个瑕不掩瑜的重制作品了。

继承与瑕疵

// 开场动画截图

牧场物语虽然开创了这种“继承牧场后，种田养殖谈恋爱走向人生巅峰”的游戏模式，但是实际上牧场物语类的游戏一直都是小众品类。唯一将这个类型发扬光大的反而是一个人开发完成的《星露谷物语》，持续不断的更新和优质的社区不断的为这个游戏注入新鲜血液，但是作为一个怀旧的老逼，这里就暂时不谈星露谷的优秀。

牧场物语这个类型实际上没有什么目标，简单的来说，这个游戏就是塞给你一个农场，你在里面想做什么都可以。你可以种地养殖，也可以谈恋爱结婚，没有任何任务目标，只有作为玩家自己给自己的目标。因此，这个游戏的玩家会相对更极端一些，喜欢玩的会感觉这个游戏非常忙，这里要种田养殖，那里要钓鱼下矿，还需要给村民送礼物，每天的时间都不够用；而不喜欢玩的会感觉每天都无所事事，这田种不种好像都没什么区别。

回归牧场物语的游戏本身，游戏的机制其实还是比较简单的：采集、种植、养殖来获取资源；烹饪、加工来生产半成品和成品；将这些资源出售转化为金钱，或是作为礼物获取好感度。底层逻辑很简单，但是在游戏体验中并不是很好。

无法满足的种田基因

风之集市这作的种田体验是比较差的，土地开满之后，总共只有4 * 6 * 9块地。其中有一块在地下室，季节是由放置的太阳来决定的，但是太阳又无法取下。最受影响的就是茶叶，根据季节的不同，茶叶的产物是不同的，因此有春茶、夏茶、秋茶之分，而玩家期望的一般只有收益最高的秋茶。在地下室里有多个太阳的情况下，茶叶的产出是随机的，不能保证稳定的秋茶收益。

田地的数量也不太够，风集这作的种子数量相当多。除去6种蘑菇和4种牧草，每个季节都有大约20种左右的作物和果树需要种植。果树和农作物都需要占用耕地，哪怕仅仅只是为了收藏，每小块地只种一种作物，也只是勉强够用，更不用说要赚钱或者一些大需求量的产品而进行大批量的种植了。

// 农场截图

风集在农场整体环境的设计上还是非常棒的，超大的牧场，一直在运转的风车和旁边的梯田都很让人喜欢。但是在游戏性上就差了一些，特别突出的一点就是两个动物小屋距离超级远，分别在草场的两侧，每天去捡鸡蛋都需要跑很远，再配上那不同星级的鸡蛋和有限的背包，来回两趟相当折磨。农田也是一样的问题，为了方便剧情的拓展和美观将田地划分成了多个小块，导致每块田都偏小，导致种田的过程非常“不爽”。

重复劳作的养殖问题

虽然牧场物语系列中的鸡牛羊都很可爱，但是M社对于动物养殖这部分的内容有着让人很难以理解的固执，比如每一作都有并且几乎是强制的”抚摸动物“。游戏中，抚摸动物会让动物提升好感值，进而产出更优质的产品。

在旧作中，好感度高的动物可以产生更高级的产品，比如小瓶牛奶，中瓶牛奶和大瓶牛奶。而本作中，好感度高的动物产生的产品星级会更高，抚摸也会提升特殊产品的产出概率，配合芝士大蛋糕的抚摸提升稀有副产品出现概率 lv4，几乎可以做到每天一份黄金产物。

// 芝士大蛋糕截图

这个设计本身其实还挺不错的，但是问题在于抚摸动物这个操作重复次数太多了，并且这个动作并没有进行后续的优化。游戏总共可以养9只宠物，10只鸡，10只牛/羊/羊驼，也就是你每天需要至少抚摸29次。这还不包括这些动物需要的额外的拥抱和刷洗。

对比一下种植，种植每天需要的浇水可以通过升级水壶来增加范围，最高级的水壶可以通过一次跳跃浇54格的田地；如果不在意耕地占用和稀有副产品减少的问题，还可以使用自动洒水器变成完全的自动化。而养殖这块保留了M社的初心，从GBA开始就是这样，到现在仍然是这样。唯一进步的点就是你的宠物会帮你放牧，然后就没了。

优秀的风车和集市

毕竟本作的名字就叫风之集市，核心自然是对应的风车和集市。风车的效率和风速挂钩，风量越大，风车运行效率越高。原作在NDS上时，还可以通过往麦克风吹气的方式，加速风车转动来提高运行效率。而集市是每周六才开始，配合上物品的新鲜度的设计，玩家需要更多的考虑金钱的流转周期，货物的储存等问题。相比于传统的成品丢到收货箱然后第二天领钱的设计，并不好判断孰优孰劣，但是风集的集市设定确实还算有趣。

关于赚钱

模拟经营类游戏里最重要的一点就是赚钱。而这类游戏的通用思路都是：素材 -> 半成品 -> 成品，即购买素材，进行加工，然后再出售成品的思路。在这个思路下，玩家会支出两种成本，分别是金钱成本、时间成本和体力成本。金钱成本主要是原材料的购买，比如购买农作物的种子，荞麦粉需要先从商人那里购买荞麦等。而时间成本主要在于植物生长的周期，风车加工原材料耗费的时间。体力成本则是人物活动需要消耗的体力，因为在前中期的体力是有限的，即使有恢复体力的食物，也仍然需要消耗宝贵的素材来制作。实际游戏过程中，素材、半成品、成品都可以作为出售的商品，销售策略的选择还是要根据实际成本来决定。

另外值得一提的是，本作产品的价格和星级的关系并非直线，而是一个星级越高越陡峭的曲线。0.5星产品的价格为基准的的话，1星产品价格为105%，而6.5星产品价格是250%，7星产品的价格是300%。因此后期的赚钱流程中应当以提高星级作为目标。

除此之外，风集由于定期集市的特点，提供了热门趋势的词条，对应的产品售价会提高30%。而物品新鲜度的下降会导致价格下降；不过游戏流程体验下来，在赚钱的生产周期中，很少会出现物品新鲜度下降得非常多的情况。更多的是囤积给村民的好感度礼物，长期带在身上导致新鲜度下降。

个人是在第一年秋季的时候达成了世界级集市，这里也会按照我自己的流程经验来划分游戏的前中后期来描述。

前期赚钱的思路

游戏前期玩家拥有的金钱和资源都很少：田地数量较少所以应避免选择长周期的产品，比如果树；工具等级较低的情况下，不应该考虑星级带来的收益。因此前期赚钱的东西选择就比较简单了：能进行二次加工，获取周期短或是获取成本低的商品。

前期最容易获取的东西就是野花和野草了，并缺春季也是野花野草最多的时候。野花野草都可以做香水，但是野草可以做沙拉来弥补前期匮乏的体力。所以推荐是使用野花做香水、野草做沙拉恢复体力。

// 野花照片

在第一次正式的集市期间，就可以从商人那里购买鲜花种子。每个季节都有固定的三种鲜花用来做花束，花束的售价高达1000元，而鲜花的生长周期仅需要3天。所以囤积一笔启动资金购买鲜花种子做花束是一个相当优质的选择。

// 春季花束

另一个稳定的收益就是牧草种子，牧草在种植之后不需要浇水，生长后还会自动扩散。收割还有概率掉落种子，可以重复的种植。在前期也是一个非常不错的收入来源。

// 牧草

但是实际在这个游戏中，正常的种植都没有亏损一说。前期种萝卜之类的作物，然后直接到集市上售卖也完全没有问题。只是类似果树这样的，生长周期长，回本缓慢，会让前期变得更加折磨。

中期赚钱的思路

中期也就是有一定的经济基础的时期。田地没开满但是也够用，动物能保证少量的牛奶、鸡蛋产出，风车也都已经全部解锁。这个时候就可以按照”素材加工半成品，再生产成品后销售“的思路进行赚钱了。由于中期的仓库位置紧缺、菌种木桩和蜂场的扩展以及土地升级之类的大量消耗金币的支出，所以中期应该以成本为优先：保证每次集市后都有足够的资金进行上述的拓展操作。既然要节约成本，那么考虑的作物就应当是能连续采摘的作物、日常都能采集的采集物和每天稳定产出的牛奶、鸡蛋。

可以仍然延续前期的赚钱思路，继续使用香水、花束赚钱，这些都是相当稳定的收入来源，仅需要使用风车的加工就可以做到。但是由于逐渐要开始进行农具的升级，生产香水、花束的红色风车会被占用进行木材和石材的加工，所以需要考虑出售其他的商品，在风车的空档期进行销售。另外，牧草由于它几乎0成本的特性，可以一直销售。

蓝色风车有一个稳定的收益就是做年糕，餐厅直接购买米饭，然后丢到蓝色风车中加工成年糕，就是一个稳定的时间转化成金钱收益的流程。

// 年糕

其次就是做首饰，首饰的加工消耗的时间非常非常多，但是由此产生的收益也非常大。大部分首饰都可以做到成品是素材的两倍价格，而做首饰需要的素材都是日常挖矿的产物和副产物。当我升级到银农具时，铜矿就几乎没有什么用了，就可以全部制作成首饰进行售卖，后续的金农具、山铜农具也是类似的道理。

// 首饰

另外就是做料理，料理的加工是这个游戏中收益最大的环节，因为做料理的过程没有任何成本，即可以瞬间把素材变成成品并且升值。在中期这个阶段，考虑的仍然是节约成本，因此料理主要选择了以连续作物、牛奶鸡蛋为主的配方，比如蛋包饭，仅需要红色风车加工出黄油，配合做年糕富裕的米饭就可以完成。

// 蛋包饭

中期其实还有一些本身是很优质的料理，但是配方并不一定能在当时就解锁。比如赏月丸子和一些小麦粉相关的料理，需要通过集市解锁新的商店，再去商店购买对应的食物吃下去，之后才解锁配方。这些配方我都是在后期有更优质的赚钱手段后才获得。所以这里也不好去展开讲解

对于作物，夏季的水果和秋天的茶叶都是很不错的收益。但是由于当时我并没有注意到果茶这个选项，在夏季把水果都卖了或者做了种子，然后秋季仅做来一些普通的红茶，但是收益仍然比较可观。

后期的赚钱思路

后期的基本上就是料理配方基本上都解锁，农田拓展完毕，斧头到山铜级，也就是处于一个想做什么都能做的阶段。因此后期赚钱的思路主要在于两点，最大化利用素材的星级和减少时间成本。由于风集这作的农作物和畜牧产品想要到达7星必须经历两次评选冠军后才行，而售价更高的品牌化还需要更多一次，因此需要做相当程度的提前规划才可以。正常流程中前几个月都是处于一种摸索的状态，但是实际上这些赚钱的流程，并不是高星才能赚钱，而是高星赚钱更多。

首先是但凡了解过一点攻略都应该听说过的荞麦冷面和格雷麦饼。荞麦冷面的素材需求仅为小麦粉和荞麦粉，小麦粉是由种植获得的小麦再红色风车加工，荞麦粉则是从商人购买后再在蓝色风车加工获得。格雷麦饼的素材是土豆、荞麦粉和油，与荞麦冷面非常类似，区别就是土豆无需在红色风车加工，但是需要从商人额外购买油。

// 荞麦冷面 格雷麦饼

这两个商品的共同特征都是产出的星级仅和农产品的星级相关，因此提升非常容易。而风车加工的素材仅从商店购买，不会和农作物抢占田地空间；最后料理的制作无需消耗时间，有多少素材就有多少成品。因此，只要你能得到7星的小麦或是土豆，就可以获得7星的荞麦冷面或格雷麦饼。

基于荞麦冷面的思考

从荞麦冷面和格雷麦饼中我们可以看到，风集这个游戏中获取素材其实是有几条并行的路线：

  • 种植
  • 畜牧业养殖
  • 周期收益，即采蘑菇和收蜂蜜
  • 采集，包括钓鱼、采集野花野草、采矿、捕虫
  • 商店购买，包括每天都可以购买的村里的商店和每周可以购买的集市商店

而我们的目标是高星商品，那么就需要容易变成高星的素材。

  • 种植：7星作物至少需要获得两次冠军，多季度作物意味着至少两季度，而单季度作物更是需要两年的时间

  • 畜牧业：不仅需要每天抚摸拥抱刷洗，还需要获得两次冠军，是我觉得最麻烦的

  • 周期收益：将产物直接喂给小矮人换星级，蜂蜜产出多所以提升非常快，后期会溢出大量，可以直接视作0成本素材。蘑菇虽然产出少一些，升星也慢一点，还需要占用一部分蓝色风车资源去做种子，但是也是后期也是溢出的，所以也可以看作0成本

  • 采集：升星方法很简单，从矿物店老板那里买石头然后喂给小矮人，几十万就可以达到7星，素材需要每日采集，但是收益稳定。钓鱼的话只能慢慢钓鱼然后喂小矮人，会花很多时间。内部排序的话，采矿 »» 钓鱼 > 野花野草捕虫。采集里的东西只有采矿这部分内容是适合赚钱的。

  • 商店购买：没有星级就跳过，可能会占用一些风车资源。

风集中素材获取部分都是需要时间成本的，无论是种植——消耗大量时间成本后获取大量素材，还是其他的方式——消耗少量时间成本后获取少量素材。而把素材加工成成品只有两个方法，一个是风车，另一个是料理。其中风车是需要时间成本的，而料理不需要任何成本。因此，要获取更大的收益，就需要让这两个时间消耗并行的处理。

比如格雷麦饼：种植土豆期间，蓝色风车加工荞麦粉，商店购买并囤积油，就可以保证土豆产出的时候就可以直接销售格雷麦饼。所以格雷麦饼这样生产产品的形式非常适合中期时候，每个阶段都缺钱的阶段。但是格雷麦饼是在地区级集市才解锁，也就是意味着解锁这样的产品的时候，玩家一般已经有一套自己习惯的赚钱流程，比如我已经用秋茶和首饰赚了一大笔。

荞麦冷面的反思

上面说了这么多时间成本的事，还特意绕开了荞麦冷面只说了格雷麦饼，大家都会意识到一个问题：这个游戏并不是一个极度缺乏金钱的游戏，也不是一个有“还债”压力的游戏，完全没有必要压榨每个集市去赚很多钱，休闲一周并不会产生什么影响。

但是我们游玩的时候会发现，风车经常会出现加工栏位已满，导致我们不能推进其他的计划目标。也就是说，风车的时间成本，在游戏中是以风车的容量为体现的。由于本作中工具升级需要大量的木材，而木材不仅需要经过三个风车轮流加工，还耗费大量时间。而作物的种子、肥料，蜂蜜的加工等等，都是在占用风车的工作时间。因此我们的赚钱之路上，就需要更多的优化风车的利用率。

好在风集有宝珠，它的作用就是优化风车的使用体验。里面比较值得关注的是：

// 宝珠

  • 蓝色：缩短加工时间
  • 黄色：单次加工每生产5个就增加1个产出
  • 橙色：单次加工的产品上限提升至30个
  • 绿色：多6条生产线，总共变成12条

蓝色就是朴实无华的缩短时间，但是需要多跑几次风车，在有了飞行石之后是一个比较不错的选择。

黄色是增加产出，适合给种子加工、首饰加工和不当人的木材需求量来增加产出。

橙色适合收获周期长的产品，比如在红色风车里加工小麦粉。

绿色是最泛用的珠子，增加生产线也意味着不仅降低时间成本，还提供了更大的容量。

根据上面的珠子，我们就可以比较简单的拼凑出一个，红色风车使用橙色珠子加工小麦粉，蓝色风车使用绿色珠子的空余位置加工荞麦粉，进而就可以做到荞麦冷面。

红茶和果茶

三个风车之间的功能不同，因此使用率上也有非常大的差距。红色风车是最先解锁的，里面的产出也是最开始的基础作物的半成品；蓝色风车是第二个解锁的，里面是非常重要的种子加工、蜂蜜加工、还有荞麦粉；黄色风车是最后解锁的，里面的内容就比较高级和昂贵，比如茶叶、果茶之类。所以游戏在解锁这三个风车之后，我们对于风车的使用率是这样的：蓝色风车 » 红色风车 > 黄色风车。这也是为什么我会吧绿色宝珠给蓝色风车的原因，因为它的使用率太高了。

一旦蓝色正在加工蜂蜜、种子和木材的时候，蓝色风车就只能提供很少的位置加工荞麦粉。因此我们可以把目光转移到另外两个风车。比如红色风车的小麦粉和黄色风车的茶叶。

// 果茶

其中黄色风车的果茶是很典型的长周期高利润产品。通过任意的水果和秋茶，在黄色风车加工出茶叶后再进行料理即可。水果可以是春季草莓、夏季西瓜、秋季蜜瓜，也可以是苹果、柠檬哪些书上长的，而秋茶是仅能在秋季生长的茶叶，茶叶是连续作物，一波种子可以产生非常非常非常多的秋茶。两者组合加工后，就可以产出高额利润的果茶了。更重要的是，在制作果茶的期间，红色风车和蓝色风车仍然可以制作荞麦粉、小麦粉等产物，不会影响到荞麦冷面的产出。

脱离农作物的首饰

游戏的过程中并不一定能注意到这些收益非常高的商品，但是任何玩家都会有升级工具的目标，也就有采矿的需求，因此肯定能注意到风车里有几个经常亮起来，还特别昂贵的首饰商品。牧场物语类的玩家基本上都会有一定程度的挖矿习惯，特别是风集这作中，矿物的掉落是概率的，变相强迫玩家每天去挖矿获取升级素材。因此，将溢出的矿物在空闲的风车中加工成首饰，是一个很自然而然但是又达到了我们高收益目标的行为。

//首饰

脱离风车的矿产

很快我们就会发现，首饰的加工时间实在是太太太长了，动辄好几天的加工时间让我们在有了使用风车的需求的时候也无法使用风车。而矿石本身的售价就相当高，银矿基础售价是800，金矿的基础售价是2000，更重要的是鸡蛋芝士面包船，食物效果是采集的数量翻倍。在山铜工具后，金矿的出率都非常高了；根据个人体感而言，配合食物buff每天仅采集两轮矿物，每周收获的金、银、铜矿都有小一组，也就是(500+800+2000)*300%*99 = 980,100算上集市装扮和小矮人加成，以及山铜和首饰石头的掉落，每周稳定收入100万不是什么大问题，而面包船的基础售价是10000元左右，所以只要面包船期间额外掉了5个金矿就已经回本了，更不用说风集的物品刷新特性是5点和6点各一次：意思就是我们5点之前采完一轮矿、5-6点之间再采一轮、6点之后还能采一轮；一天三轮矿物肯定赚得盆满钵满。

// 鸡蛋芝士面包船

如此大量的矿物获取量，没必要为了首饰的额外收益去挤占风车的运行。而核心中的核心就是鸡蛋芝士面包船，它的素材是小麦粉、极品芝士、极品酸奶、极品黄油。这三个极品素材，都是由普通牛和水牛概率产出的极品牛奶和极品水牛奶，在黄色风车中加工获得的，而提升星级所使用的可以是鸡蛋、米粉、砂糖、盐、橄榄、南瓜籽油，这里面的鸡蛋、砂糖之类都是非常好获得的东西。

所以难以获得的只有每天概率产生的极品牛奶和极品水牛奶。注意力惊人的我们可以注意到前文提到过的芝士大蛋糕这个物品，有着抚摸提升稀有副产品出现概率 lv4的优秀技能，并且这个物品在每周集市中可以买到3个。而4级的技能基本上能确保每次抚摸大概率产出极品牛奶和极品水牛奶，考虑到做衣服占用了两只羊和两只羊驼，所以6头牛（2普通牛和4水牛）的比例，保证了每周能够至少6桶极品牛奶和12桶极品水牛奶；6天需要消耗的6个面包船正好是需要6个极品牛奶的半成品和12个极品水牛奶的半成品。由于牛的产出数量是随机的，所以收获的牛奶基本上会大于消耗，也就意味着在获取量鸡蛋芝士面包船的配方后就可以开始进行双倍矿产采集。除此之外，小麦粉是我们前文提到过的荞麦冷面的必备素材，如果我们之前使用了荞麦冷面赚钱，那么这时候就可以顺手再做一些面包船。

而溢出的素材怎么办，芝士大蛋糕的素材要求是小麦粉、极品鸡蛋、极品水牛奶、芝士。溢出的水牛奶和我们暂时用不到的鸡蛋就可以在这里进行制作，创新栏可以使用后期必定会溢出的蜂蜜进行加星。另外巧克力大蛋糕也是同样的效果，素材不需要芝士而是替换成了巧克力。但是由于它是甜点大赛奖励，无法购买必须自己制作，我作为一个懒人还是选择了芝士大蛋糕。

到这里，我们的种植、畜牧、采集、加工这些本来本来没有什么交集的就汇聚到了一起：

小麦种植 -> 红色风车加工成小麦粉

购买荞麦 -> 蓝色风车加工成荞麦粉

畜牧养殖 -> 芝士大蛋糕获取稀有副产物 -> 黄色风车加工成极品芝士、酸奶、黄油 -> 制作鸡蛋芝士面包船 -> 挖矿

副产物进一步的优化

根据鸡蛋芝士面包船的思路，我们可以看到我们还有鸡蛋、蜂蜜、蘑菇、普通牛奶这些都是还没用上的。而小麦和荞麦的思路确实会占用大量风车且收益有限。那么我们是否还可以继续优化它呢？

首先我们来梳理一下我们还富裕的产品：

小麦、鸡蛋、极品鸡蛋、普通牛奶、蜂蜜、松茸、大王菌菇。对于必要的鸡蛋芝士面包船我们需要的小麦并没有很多，所以我们可以视作我们还溢出了大量农田可以任意由我们分配。

其次我们需要观察一下那些高价值的产品，根据成品的基础售价减去素材的基础售价，我们可以粗略的获得一些高利润的料理top10：
  • 黄金特调茶（11879）

黄金特调茶会抢占风车，性价比不如首饰。而不做首饰的话完全可以考虑按量取胜的果茶，所以不考虑

  • 终极咖喱（9922）

终极咖喱需要极品咖喱、巨大番茄、指甲辣椒、极品酸奶。种植里的稀有副产物并不像养殖那样容易获得，所以也不考虑

  • 鸡蛋芝士面包船（8673）

鸡蛋芝士面包船本来就是我们的目标，而且本身价值就非常高，溢出了直接售卖是完全没有问题的

  • 特调果汁（9511）

特调果汁非常值得一说，因为他的素材里只有草莓是需要种植的，其他的都是由果树获得。虽然我们的果树基本上都只能获得5星产出，但是春风、夏风、秋风果汁都可以通过蜂蜜作为配料来提星。最终产出的特调果汁也可以通过奢华的蜂蜜来提星，这些蜂蜜是我们溢出非常多的东西。所以特调果汁完全值得考虑制作

  • 农家早餐（5927）

农家早餐的素材是马铃薯、洋葱、极品芝士和极品蛋黄酱，但是升星的配料是不太容易获取的各种油。它是一个比较好来消耗极品蛋黄酱的渠道，而且马铃薯和之前的格雷麦饼也很适配，但是问题在于我并不期望抚摸每一只动物，极品鸡蛋的获取量并不多。当积攒了一定量的鸡蛋后，可以再通过这个料理去消耗掉。

  • 水果蛋糕杯（4767）

水果蛋糕杯的素材是古早蛋糕、极品乌骨鸡蛋、极品酸奶、水果。而古早蛋糕的素材是小麦粉、乌骨鸡蛋、蜂蜜。根据素材我们可以看得出来，水果蛋糕杯是很容易获得7星产品，它还消耗掉了我们溢出的鸡蛋和小麦粉，任意的水果也和前面的特调果汁很搭。只不过它值消耗了乌骨鸡蛋和并没有消耗普通鸡蛋。

  • 炒米粉（3599）

炒米粉的素材是米粉、大王菌菇、卷心菜、胡萝卜还可以通过酱油、油来升星。在获得松茸后，大王菌菇就是稳定的素材来源，基本上可以保证每三天获得三个。如果我们按照炒米粉的需求来种植水稻、卷心菜和胡萝卜，那么炒米粉就是一个非常值得考虑的料理

  • 极品咖喱（2274）

极品咖喱的问题和终极咖喱一样，都需要种植部分的稀有副产物。所以我们就直接跳过。

  • 芝士通心粉（2127）

芝士通心粉的素材需求为小麦粉、芝士、黄油、蛋黄酱。完美的配方，一下子就把我们溢出的普通素材消耗了，还提供了非常高额的利润。

  • 凤梨派（1825）

凤梨派的问题也在于种植的稀有副产品：宝石凤梨，所以也不考虑。

综上，我们期望的是

特调果汁：消耗水果和蜂蜜，无需浇水，占用蓝色风车

水果蛋糕杯：消耗乌骨鸡蛋、极品乌骨鸡蛋、水果

炒米粉：消耗大王菌菇，但是需要种植水稻、卷心菜、胡萝卜，占用红色风车

芝士通心粉：消耗普通牛奶、普通鸡蛋，占用红色风车

寻找剩下素材的成品，我们还剩下极品鸡蛋和松茸。

  • 松茸只有松茸饭和烤松茸两个料理可以制作。那么松茸饭就是最佳选项了，素材是米、松茸、酱油。此时它又正好契合了炒米粉需要额外种植的水稻。

  • 极品鸡蛋可以考虑两个消耗途径，一个是农家早餐，但是需要额外种植一点马铃薯和洋葱，另一个就是前文中提到过的巧克力大蛋糕或者芝士大蛋糕，也就是提升抚摸产生稀有副产品概率的那个。其中巧克力大蛋糕的利润较低只有871，还需要消耗极品牛奶；而芝士大蛋糕利润较高1651，但是消耗的是极品水牛奶和芝士，实际中完全可以考虑根据产品溢出情况来选择。我个人更喜欢制作蛋糕这作方式，因为对于特调果汁，他种植果树需要消耗相当大量的种植资源。

进一步细分种植需求

到此我们把所有的素材都消耗完了，我们还需要进一步细分我们的种植需求，养殖可以按照我们之前的2牛4水牛，任意数量的鸡和乌骨鸡（乌骨鸡为主）来进行。

特调果汁是由春风、夏风、秋风果汁合成的，春风需要草莓、橙子、樱桃；夏风需要香蕉、桃子、柠檬；秋风需要蓝莓、葡萄、苹果。也就是9*9块地。由于草莓是需要春季种植的，所以需要放置在地下室中，而且溢出的草莓可以用在水果蛋糕杯上。

剩下的15块地，我们需要种植水稻、卷心菜、胡萝卜、小麦。由于大王菌菇收益虽然稳定但是数量不多，所以卷心菜和胡萝卜哪怕各自只种植一块地都大概率会溢出，所以我们将其视作一块地（实际上是应季种植一块地，供给一年使用），剩下14块地的分配就可以很随意了。小麦、水稻、水果、茶叶按需分配就好了。其中提到茶叶的原因是：西瓜茶的利润有1538，但是代价是抢占黄色风车做首饰的生产。

划分风车职责

在新的计划中，我们需要制作芝士、黄油、蛋黄酱、小麦粉、米粉、酱油，这些都是红色风车的职责。风车需要加工的物品品类增加，因此更适合红色风车的宝珠就是绿色宝珠，增加到12条生产线。

新的规划中蓝色风车的使用量就大幅度减少了，主要是蜂蜜加工和反种。因此橙色宝珠（增加单挑产线到30个成品）或者黄色宝珠（单次加工5个多一个成品）两个宝珠都非常适合。橙色宝珠主要是用来加工蜂蜜，而黄色宝珠用来反种。

最后黄色风车的功能就是制作几个极品素材，可以安装紫色宝珠把溢出的产线用来制作最贵的首饰。平时售卖可以使用餐厅类型的集市装扮，待首饰攒够数量后就可以使用风车类型的装扮再集中售卖。

最终的赚钱思路

实际上说了这么多，最后赚钱的思路还是根据素材获取的短板来定的。资源获取中有两个比较明显的短板，一个是种植的稀有副产物，另一个是风车的加工生产线。而解决办法就是避免使用种植的稀有副产物，以及风车加工的产品尽量以高价值为主。

// 集市赚钱截图

  • 要制作的内容：鸡蛋芝士面包船、特调果汁、水果蛋糕杯、炒米粉、芝士通心粉、松茸饭、巧克力大蛋糕

  • 要种植的内容：所有种植内容都不需要稀有副产物，可以使用洒水器自动进行

    • 水果：草莓、橙子、樱桃、香蕉、桃子、柠檬、蓝莓、葡萄、苹果

    • 作物：水稻、卷心菜、胡萝卜、小麦

  • 要养殖的内容：2牛，4水牛，乌骨鸡为主，普通鸡为辅助。每天吃大蛋糕抚摸动物。

  • 要采集的内容：固定的蜂蜜采集和松茸种植，每天下午3点进行面包船的双倍采矿

  • 要加工的内容：

    • 红色风车：芝士、黄油、蛋黄酱、小麦粉、米粉、酱油
    • 蓝色风车：蜂蜜，种子
    • 黄色风车：极品芝士、极品黄油、极品酸奶，首饰

总结

赚钱其实就这些了，与其说是思路，不如说是玩这种类型的游戏的自然而然的习惯。基本上都是以摸清游戏中的素材获取和加工手段后，再查询资料找到个人认为的最优解。哪怕前文中提到了我自己特别讨厌每天抚摸动物这种操作，但是想要达到高利润后这种行为反而是必要的。

然而这个游戏是自由的，可以不去功利化的追求最高的收益，可以以任何自己喜欢的方式来游玩。我可以每个夏季种满西瓜，秋季种满秋茶，只做西瓜茶也完全可以完成目标，这也是牧场物语类型的游戏里最好的地方，每个玩家都可以享受属于自己的生活，可以繁忙，可以清闲，可以为了达成世界化集市的目标，也可以是看着周六集市结束后榭露芙提着裙摆跑回家的样子，或是香月离开寺庙回归尘世享受生活。

风集的终盘内容其实相当欠缺，以角色剧情为例，虽然子供向剧情并没有期待它能被写得很好，但是它应该是完整的。婚前婚后除了角色居住在农场之外毫无改变，并且角色之间互动也没有改变；你会看到你的妻子/丈夫和M社钦定的cp在每个节日的时候待在一起，而你是个为了节日冠军而放弃了一切的农场主。风集的地图很小，当你熟悉了一切内容的时候，游戏其实就已经没有新内容了，做的只是日复一日的赚钱。相比出圈的星露谷，这也是星露谷最大的优势，庞大的社区，自定义的mod，只要你想就总有东西可以完。

好像这个游戏也没什么可以说的了，毕竟牧场物语这个系列似乎也是这样。写着写着、玩着玩着突然感觉好像没有什么继续下去的必要，可以放下手柄出去转一转，steam库里还有很多好玩的。
